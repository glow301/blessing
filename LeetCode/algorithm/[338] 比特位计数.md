## 比特位计数
> leetcode - 338

### Description
给定一个非负整数 num。对于 0 ≤ i ≤ num 范围中的每个数字 i ，计算其二进制数中的 1 的数目并将它们作为数组返回。

### 示例
    输入: 2
    输出: [0,1,1]

    输入: 5
    输出: [0,1,1,2,1,2]

### 分析
可以观察一下规律
|数字|二进制形式|1 的个数|
|--|--|--|
|1|1|1|
|2|10|1|
|3|11|2|
|4|100|1|
|5|101|2|
|6|110|2|

可以发现
* 一个数字 i，和它的一半 i/2，除了末位之外完全相同。
* 用 `dp[i]` 表示数字 i 的二进制数中 i 的个数，那么 `dp[i] = dp[i>>1] + (1&1)`。

### Code
```go
func countBits(num int) []int {
    ans := make([]int, num+1)
    for i := 1; i <= num; i++ {
        ans[i] = ans[i>>1] + (i&1)
    }
    return ans
}
```
