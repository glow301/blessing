## 一只小蜜蜂
> HDU - 2044

### Description
有一只经过训练的蜜蜂只能爬向右侧相邻的蜂房，不能反向爬行。请编程计算蜜蜂从蜂房a爬到蜂房b的可能路线数。 
其中，蜂房的结构如下所示。 
<img src="https://vj.e949.cn/db4917bb4b1bd4e9ccc285a9597f8e31?v=1543133996">

### Input
输入数据的第一行是一个整数 N,表示测试实例的个数，然后是 N 行数据，每行包含两个整数 a 和 b (0 < a < b < 50)。 

### Output
对于每个测试实例，请输出蜜蜂从蜂房 a 爬到蜂房 b 的可能路线数，每个实例的输出占一行。 

### Sample Inupt
2  
1 2  
3 6  

### Sample Output
1  
3  

### 分析
* 用 `dp[i]` 表示从起点走到第 i 个位置，可能的路线数。
* 可以发现 `dp[i] = dp[i-1] + dp[i-2]`，也就是斐波那契数列，对于给定的起点 a 和终点 b，a 到 b 的答案其实就是 `dp[b-a]`

### Code
##### C++
```cpp
#include<cstdio>
#include<algorithm>
#include<cstring>

using namespace std;

const int MAXN = 50;
long long dp[MAXN];

int main() {
    int num = 0;
    scanf("%d", &num);
    while (num--) {
        int a, b;
        scanf("%d %d", &a, &b);
        dp[0] = dp[1] = 1;
        for (int i = 2; i <= MAXN; i++) {
            dp[i] = dp[i-1] + dp[i-2];
        }
        printf("%lld\n", dp[b-a]);
    }
}
```

##### GO
```go
package main

import "fmt"

const MAXN = 50

var dp [MAXN]int

func main() {
    line := 0
    fmt.Scanf("%d\n", &line)

    dp[0], dp[1] = 1, 1
    for i := 2; i < MAXN; i++ {
        dp[i] = dp[i-1] + dp[i-2]
    }

    for i := 0; i < line; i++ {
        a, b := 0, 0
        fmt.Scanf("%d %d\n", &a, &b)
        fmt.Println(dp[b-a])
    }
}
```
